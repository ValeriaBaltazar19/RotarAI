# Rotar AI - The Ultimate AI Customer Service Companion  

Este proyecto es una aplicación web que ofrece un asistente de servicio al cliente impulsado por inteligencia artificial, utilizando la tecnología de chatbot y el soporte de GPT-4.

## Descripción

Rotar AI es una herramienta de comunicación diseñada para ayudar a tus clientes, empoderar a tu equipo y reducir tu carga de trabajo. Utiliza la tecnología de chatbot impulsada por GPT-4 para proporcionar respuestas rápidas y precisas a las consultas de los clientes.

## Características

* **Chatbot impulsado por IA:** Utiliza la tecnología de GPT-4 para proporcionar respuestas inteligentes y precisas.
* **Interfaz de usuario intuitiva:** Diseño limpio y fácil de usar para una experiencia de usuario óptima.
* **Logotipos de empresas:** Muestra logotipos de empresas reconocidas que utilizan Rotar AI.
* **Tutorial:** Incluye una guía completa para ayudar a los usuarios a comenzar con los chatbots.
* **Animaciones:** Logotipos de empresas con animaciones de desplazamiento horizontal.


## Tecnologías utilizadas

* HTML5
* CSS3
* Bootstrap (opcional)
* JavaScript (para el chatbot y las animaciones)


## Uso

* Navega por las diferentes secciones de la página para conocer las características y beneficios de Rotar AI.
* Utiliza el chatbot para interactuar y obtener respuestas a tus preguntas.
* Consulta el tutorial para aprender a utilizar Rotar AI.
